# fullcalendar-nylas


## Project Setup

```shell script
npm install
```

### Compile and Hot-Reload for Development

```shell script
npm run dev
```

### Compile and Minify for Production

```shell script
npm run build
```

### Start Flask Server
```shell script
flask run
```

## Python Packages
- flask (https://flask.palletsprojects.com/en/2.2.x/)
- flask-cors
- requests
- python-dotenv
